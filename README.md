# bt_api_coinswitch

CoinSwitch exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_coinswitch
```

## Usage

```python
from bt_api_coinswitch import CoinSwitchExchangeDataSpot, CoinSwitchRequestDataSpot
```

## Features

- REST API support
- API key authentication (x-api-key header)
- Symbol format: `BTCINR` (pair string)
